# OCR.Barcode-recognition

# Стек технологий
Python, PyTorch, torchvision, Levenshtein, matplotlib, pandas, numpy, tqdm

# Описание проекта
Данный проект реализует систему распознавания штрихкодов на изображениях с использованием архитектуры CRNN. В системе применяются предобученные модели ResNet50, адаптированные для работы с grayscale изображениями, а также метрики WER (Word Error Rate) и CER (Character Error Rate) для оценки качества распознавания.
Проект включает подготовку датасета, обучение модели, оценку и визуализацию результатов.

# Реализованы:
- Воспроизводимость результатов с помощью установки seed и режима детерминизма
- Метрики WER и CER для оценки качества
- Класс Dataset для обработки изображений и текстовых меток
- DataLoader с кастомной функцией collate_fn
- Архитектура CRNN с ResNet50 в качестве энкодера и GRU в качестве декодера
- Обучение модели с использованием CTC Loss и cross-entropy
- Визуализация предсказаний и ошибок по позициям штрихкодов

# Цель проекта
Создать точную систему распознавания штрихкодов и проанализировать влияние различных настроек модели и метрик на качество.

# Основные компоненты системы

- Датасет: CSV-файл с разметкой (df_ann) и изображения штрихкодов
- Предобработка изображений: Resize, Normalize, преобразование в grayscale
- Метрики: WER и CER (расстояние Левенштейна)
- Модель: CRNN с ResNet50 (адаптированный для grayscale) и GRU
- Обучение: оптимизация с помощью Adam, использование CTC Loss
- Визуализация: примеры предсказаний, ошибки по позициям

# Модели

- Encoder: ResNet50, модифицированный для grayscale изображений
- Decoder: GRU, выводящий последовательности символов
- Обучение с использованием CTC Loss для последовательностных данных
- Варианты: использование CrossEntropyLoss в отдельных экспериментах

# Обучение и оценка

- Разделение данных на обучающую и тестовую выборки
- Обучение модели на 25 эпохах
- Метрики WER и CER для оценки качества
- Визуализация предсказаний и ошибок по позициям штрихкодов

# Результаты и выводы

- Реализована CRNN модель с CTC и Classification подходами для распознавания штрих-кодов EAN-13.
- Classification с CrossEntropyLoss показывает лучшие результаты — оптимален для фиксированной длины 13 символов
- Grayscale preprocessing критически важен — значительно улучшает качество при меньших вычислениях
- Dataset normalization обязательна — стабилизирует обучение на реальных изображениях
