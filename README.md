# OCR.Barcode-recognition

# Стек технологий
Python, PyTorch, torchvision, NumPy, pandas, Levenshtein, Matplotlib, tqdm

# Описание проекта
Данный проект реализует систему распознавания штрихкодов на изображениях с использованием сверточных и рекуррентных нейронных сетей (CRNN). В проекте предусмотрены функции для подготовки данных, обучения модели, оценки качества и визуализации результатов.

# Реализованы следующие компоненты:
- Предобработка изображений (цветные и grayscale версии)
- Метрики оценки качества: Word Error Rate (WER) и Character Error Rate (CER)
- Архитектура модели: Encoder (ResNet50), Decoder (GRU), полноценный CRNN
- Обучение и тестирование моделей с использованием CTC Loss и CrossEntropyLoss
- Визуализация предсказаний и ошибок по позициям штрихкодов

# Цели проекта
Создать универсальную и расширяемую библиотеку для распознавания штрихкодов на изображениях, позволяющую:
- Обучать модели на различных датасетах
- Оценивать качество распознавания с помощью метрик WER и CER
- Визуализировать результаты и анализировать ошибки
- Экспериментировать с различными архитектурами и функциями потерь

# Эксперименты
- Замена функции потерь (CTC Loss / CrossEntropyLoss)
- Использование grayscale изображений с нормализацией
- Замена архитектуры энкодера на версию для grayscale
- Анализ ошибок по позициям штрихкодов
